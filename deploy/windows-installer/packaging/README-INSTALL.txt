CM TechMap Installer - Quick Start (Windows)

1) Execute CmTechMapInstaller-Setup.exe
2) Open the installed app from Start Menu
3) Configure:
   - Repository URL
   - Branch
   - Frontend URL (your Vercel URL)
   - Tunnel mode (quick or token)
4) Click "Install Now"

After success, local backend services are started and ready.

If Docker Desktop or WSL are not ready, the installer will guide setup and retry.

- CmTechMapInstaller.exe
